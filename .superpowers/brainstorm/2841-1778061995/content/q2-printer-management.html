Вопрос 2: Управление принтерами

Сейчас принтеры — статичный JSON в .env. Для v1.4 нужна возможность администратору включать/выключать принтеры из UI. Где хранить конфиг принтеров?

A

Перенести в БД (таблица printers)

Полный CRUD через adminку: добавить, удалить, включить/выключить, задать IP/порт/DPI. .env остаётся только для первоначального сида. Самый гибкий вариант.

B

Оставить в .env, добавить флаг visible в БД

Конфиг принтеров (IP, порт) по-прежнему в .env. В БД только таблица printer_visibility (id, is_visible). Проще, но для добавления нового принтера нужен рестарт сервиса.
